简体中文

✨ 特性

  • 提炼自企业级移动端产品的交互和视觉风格
  • 开箱即用的高质量 Flutter 组件
  • 提供满足业务差异的主题定制能力
  • 设计工具赋能开发全链路

接入

Flutter 工程中 pubspec.yaml 文件里加入以下依赖：

dependencies:
  ws_flutter: version
